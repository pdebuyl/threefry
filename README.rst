threefry
========

.. image:: https://travis-ci.org/pdebuyl/threefry.svg?branch=master
    :target: https://travis-ci.org/pdebuyl/threefry
    :alt: Test Status

.. image:: https://readthedocs.org/projects/threefry/badge/?version=latest

A Cython implementation of the Threefry random number generator, part of
`Random123 <http://www.deshawresearch.com/resources_random123.html>`_.

Installation
------------

    pip3 install --user threefry

or

    python3 -m pip install --user threefry
